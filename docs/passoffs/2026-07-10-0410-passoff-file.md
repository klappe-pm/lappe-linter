## Lessons learned

- Rigor retro: session ran at `high` (multi-package feature build, shared contracts); obligations met via the workflow's per-feature test gates, an independent final verifier round, and my own end-to-end CLI verification before claiming done. No mismatch.
- Lesson: an inherited-context fork agent given an orchestration-status framing echoed the status back instead of executing (0 tool calls). Why: forks inherit the parent's narrative voice and can adopt the reporter role instead of the worker role. How to apply: dispatch large implementation briefs to fresh agents with self-contained prompts, or execute directly; reserve forks for tasks whose value is the inherited context itself. Recurrence: first. Enforced-by: not yet.
- Lesson: spec-assumed decoupling (F00 R2) did not survive contact with the real upstream import graph; the pure-core-alongside-upstream architecture (dec-004) delivered the same objective with zero regression risk. Why: specs written before code inspection overestimate separability. How to apply: inspect the import graph before committing to an extraction plan; prefer additive pure modules over invasive splits when a deadline is fixed. Recurrence: first. Enforced-by: not yet.
